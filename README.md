# SC2002 Turn-Based Combat

Repository for the SC2002 turn-based combat arena game project. A Java CLI turn-based combat game built with object-oriented design, modular battle mechanics, and extensible gameplay components.

## Current Setup
- `src/main/java/combatarena` for Java source code
- `docs` for UML diagrams and report files
